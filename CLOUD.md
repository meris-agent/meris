# Meris Cloud

**Meris Cloud** is a proprietary hosted SaaS (browser workspace + remote sandbox). It is **not** part of this open-source repository.

| Product | Repository | License |
|---------|------------|---------|
| **Meris Agent** (CLI, Harness, extensions) | [meris-agent/meris](https://github.com/meris-agent/meris) | MIT |
| **Meris Cloud** (API, Worker, Web) | Private — not published | Proprietary |

Use **Meris Agent** locally:

```bash
pip install meris-agent
meris init-harness .
meris run "your task"
```

Hosted Cloud signup and pricing will be announced on the product website when GA.
